# Entrevistador Médico AI

Este proyecto implementa un agente entrevistador médico AI que recopila información del usuario y genera una consulta SQL para almacenar los datos.

## Características

- Entrevista médica con lenguaje cercano al argentino.
- Recopilación de género, edad, estado de salud previo, síntoma principal y tiempo de evolución del síntoma.
- Generación de consulta SQL `INSERT` con los datos recopilados.
- Integración con la API de OpenAI para automatizar la entrevista.

## Tecnologías

- Python
- Flask
- OpenAI GPT-4o-mini
